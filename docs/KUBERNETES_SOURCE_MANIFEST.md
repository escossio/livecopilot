# KUBERNETES SOURCE MANIFEST

## Scope
Fontes oficiais permitidas para a frente `kubernetes_core`.

## Official sources
- Kubernetes official docs
  - https://kubernetes.io/docs/
- kubectl reference
  - https://kubernetes.io/docs/reference/kubectl/
- API resource definitions
  - https://kubernetes.io/docs/reference/kubernetes-api/

## Inclusion rules
- Apenas documentação oficial mantida pelo projeto Kubernetes.
- Exemplos e definições devem vir dessas páginas ou de seus subdocumentos oficiais.

## Exclusion rules
- Helm
- Operators
- Controllers avançados
- Cloud provider docs
- Terraform integration
- Any third-party tutorial or blog
