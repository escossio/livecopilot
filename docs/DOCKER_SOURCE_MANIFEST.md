# DOCKER SOURCE MANIFEST

## Scope
Fontes oficiais permitidas para a frente `docker_core`.

## Official sources
- Docker official documentation
  - https://docs.docker.com/
- Docker CLI reference
  - https://docs.docker.com/reference/cli/docker/
- Dockerfile reference
  - https://docs.docker.com/reference/dockerfile/

## Inclusion rules
- Apenas documentação oficial do Docker.
- Exemplos e definições devem vir dessas páginas ou de seus subdocumentos oficiais.

## Exclusion rules
- Docker Swarm
- Kubernetes integration
- Docker Compose avançado
- Orchestration
