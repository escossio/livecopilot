# Pipeline Lab

Pipeline Lab é o projeto irmão do LiveCopilot dedicado a orquestrar e operacionalizar o pipeline de ingestão e validação de domínios que já foi validado em C, Python e Terraform. Ele prepara os domínios, registra execuções e mantém checkpoints claros antes que o conhecimento chegue ao LiveCopilot.

O projeto está em fase inicial; por enquanto, entrega apenas a estrutura mínima para registrar domínios, runs e artefatos, sem integrações ainda. O objetivo é evoluir para um runner/API/UI dedicados quando o protocolo estiver estabelecido.
