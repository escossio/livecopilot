# Pipeline Lab - Execution Protocol

O pipeline oficial segue as etapas validadas no LiveCopilot: 1) source_policy 2) source_manifest 3) corpus_freeze 4) parsing 5) chunking 6) lexical_validation 7) chunk_refinement 8) semantic_baseline 9) semantic_refinement 10) domain_closure. Cada etapa possui um gate que exige artefatos e validações mínimos antes de liberar o próximo estágio; o protocolo garante que nada avance sem supervisão humana e registro de evidências.
