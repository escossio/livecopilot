# Pipeline Lab - Escopo do Projeto

## Propósito
Orquestrar o pipeline de ingestão e validação já validado em C, Python e Terraform, garantindo uma esteira repetível antes que o conhecimento seja entregue ao LiveCopilot.

## Escopo MVP
- catalogar domínios e seus manifests
- registrar runs e estágios cumpridos
- documentar gates e checkpoints obrigatórios
- oferecer visibilidade e logs para supervisão humana

## Fora do MVP
- integração direta com o LiveCopilot
- comparação A/B entre pipelines completos
- execução automatizada de todas as etapas
- interface rica ou dashboards complexos
